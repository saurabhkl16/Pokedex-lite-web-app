# Pokedex Lite Web App

## TechStack
- React + JavaScript + BootStrap.

## Ho to create new react app:
- First, install Node.js
- cmd prompt -> npm create vite@latest my-app
- To run web app -> npm run dev

## Ho to run the project app:
- First clone the repo.
- change directory to project.
- npm install.
- npm run dev.
